Open all linked images and videos from a page in a new window.

## Install

1. Download ZIP
2. Extract
3. Go to chrome://extensions
4. Enable Developer Mode
5. Drag folder into window

## Features

- Opens images + videos:
jpg
jpeg
png
gif
webp
avif
bmp
svg
mp4
webm

## How to use

Right-clicka preview image and select Media2Tabs.
New browser window opens with all linked media in separate tabs.
